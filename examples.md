# Skill Naming Examples

## Private Nickname To Public Name

```text
g3-control-room -> Expert Execution Upgrade
```

The nickname can remain as an alias. The public name says what the skill does.

## Vague Role To Specific Job

```text
operator -> Workspace Lifecycle Standard
```

The new name tells the agent it should organize project state, docs, logs, and versioning.

## Risky Action Naming

```text
deployer -> Approval Gated Production Deployer
```

The name itself reminds agents that external state changes require approval.
